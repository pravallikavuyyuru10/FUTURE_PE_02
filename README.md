# ClearGlow Skincare UGC Ad Content Pack

## Task
Task 2: AI Content Marketing using UGC Ads

## Product
**ClearGlow Skincare Kit** – Designed for teenagers with acne-prone, dry, and sensitive skin.

## Target Audience
- Teenagers (13–19 years old)
- Acne-prone, dry, or sensitive skin

## Brand Voice
Friendly, supportive, relatable, and teen-focused.

## Platforms
- Instagram Reels
- TikTok
- YouTube Shorts

## Content Included
This repository contains an AI-generated UGC ad content pack, including:
- Scroll-Stopping Hooks
- UGC Scripts (Problem → Solution → CTA, Testimonial, Relatable/Story)
- Call-to-Actions (CTAs)
- Social Media Captions
- Notes & Prompt Logic

## AI Tools Used
- ChatGPT (UGC script & hook generation)
- Optional tools for video or media ideas: Runway, Pika, CapCut

## Purpose
This content pack is designed to create authentic, teen-friendly social media ads that:
- Grab attention quickly
- Show relatable stories for acne-prone sensitive skin
- Encourage conversion with friendly CTAs
- Include platform-optimized captions
